## 📝 使用示例

### 在爬虫中应用

```typescript
import { getProductImagePlaceholderValue } from "../lib/product-image-placeholders";

// 爬取产品时
const product = {
  id: 'satisfyer-pro-2',
  name: 'Satisfyer Pro 2',
  subtypeCode: 'suction_pure',
  gender: 'female',
  // 如果没有真实图片，使用占位符
  imagePlaceholder: coverImage || getProductImagePlaceholderValue('suction_pure', 'female'),
};
```

### 在组件中渲染

```tsx
<ProductImage
  imageValue={product.imagePlaceholder || product.image_url}
  alt={product.name}
  iconClassName="h-8 w-8"
  imageClassName="object-cover"
/>
```
